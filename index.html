HTML

  • HTML stands for HyperText Markup Language
  • The head element contains information about the webpage
  • The body element represents the physical content shown to the user
  • The "li" element represents bullet points
  • The "h#" elements represent headings
  • I have a lot more to learn!


CSS

  • CSS stands for Cascading Style Sheets
  • CSS can be done inline, internal or external
  • The most common CSS is external
  • While HTML deals with structure and content, CSS deals with design & style
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.


Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone: retrieves an entire repo from a hosted locstion via URL
  • git log: shows all commits in a current branch's history
  • git merge: merges the specified branch's history into a current one


JavaScript

  • Adds interactivity to the website
  • Variables: containers that store values
  • Comments: snippets of text that can be added with code
  • Functions: packaging functionality you wish to reuse
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script.
